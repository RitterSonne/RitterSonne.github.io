Clasificación con KANs

Implementación de Redes Neuronales Kolmogorov-Arnold para clasificación no lineal

Python 3.x PyTorch PyKAN Scikit-Learn

Este proyecto explora el potencial de las Redes Neuronales Kolmogorov-Arnold (KAN) como alternativa a los Perceptrones Multicapa (MLP) tradicionales. Se implementa una red capaz de resolver problemas de clasificación binaria no lineal, utilizando funciones de activación aprendibles en los bordes (splines) en lugar de nodos fijos.

Arquitectura y Características

Arquitectura KAN

  • Estructura [2, 1]: 2 neuronas de entrada (coordenadas X, Y) y 1 de salida.
  • Splines Aprendibles: Uso de B-splines para capturar relaciones no lineales complejas.
  • Optimizador LBFGS: Algoritmo de segundo orden altamente efectivo para este tipo de redes.

Rendimiento & Datos

  • Aceleración Hardware: Detección automática de GPU (CUDA) o CPU.
  • Dataset Complejo: Uso de make_circles con ruido para probar la robustez topológica.
  • Visualización: Generación de fronteras de decisión en tiempo real.

Implementación del Modelo

La definición de la red KAN se realiza con parámetros específicos para controlar la complejidad de los splines:

# Definición del Modelo KAN model = KAN( width=[2, 1], # [Entradas, Salidas] grid=5, # Intervalos en la cuadrícula de splines k=5, # Orden del spline (B-splines) device=device # CPU o CUDA )

Stack Tecnológico

Python PyTorch PyKAN NumPy Matplotlib Scikit-Learn

Instalación y Uso

1. Requisitos Previos

pip install pykan torch matplotlib numpy scikit-learn

2. Ejecución

El script principal entrena el modelo y genera las visualizaciones automáticamente:

git clone https://github.com/tu-usuario/kan-classifier.git cd kan-classifier python main.py

Resultados Esperados

Al finalizar el entrenamiento, el sistema genera dos gráficos de contorno que muestran cómo el modelo ha aprendido a separar los datos:

  • Frontera de Decisión (Train): Adaptación a los datos de entrenamiento.
  • Frontera de Decisión (Test): Capacidad de generalización ante datos nuevos.

* El gráfico mostrará una región central de un color rodeada por una región externa de otro, demostrando que la KAN ha aprendido la topología no lineal de los círculos concéntricos.
