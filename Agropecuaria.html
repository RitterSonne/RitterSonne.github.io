Sistema de Gestión Agropecuaria

Aplicación de escritorio robusta para la administración integral de negocios

Estado: Finalizado .NET 8.0 Plataforma: WPF

Agropecuaria es una aplicación de escritorio desarrollada en WPF y .NET 8, diseñada para la administración de negocios agropecuarios o de venta minorista/mayorista. El sistema gestiona inventarios, facturación, clientes, proveedores, finanzas y reportes, con soporte para múltiples monedas (Pesos, Dólares, Bolívares).

Características Principales

Gestión de Inventario

  • Productos: Altas, bajas y modificaciones con soporte para imágenes.
  • Precios Múltiples: Costo, detal, mayor y especial.
  • Organización: Múltiples depósitos y categorías (grupos).
  • Trazabilidad: Control de lotes y fechas de vencimiento.
  • Alertas: Notificaciones de bajo stock o vencimientos próximos.
  • Venta Unitaria: Descomposición de productos para venta por unidad.

Facturación y Ventas

  • Facturación: Creación de facturas para Clientes y Proveedores.
  • Traslados: Movimiento de mercancía entre depósitos.
  • Multimoneda: Conversión automática (Pesos, USD, Bolívares).
  • Pagos: Soporte para Efectivo, Tarjeta y Transferencia.

CRM (Clientes y Proveedores)

  • Base de datos detallada de contactos.
  • Historial de compras y facturas asociadas.
  • Cuentas por Cobrar/Pagar: Gestión de deudas y abonos.

Finanzas y Reportes

  • Dashboard: Resumen visual de ventas y estadísticas.
  • Caja: Registro de Ingresos, Gastos e Inversiones.
  • Balances: Cálculo de ganancias.
  • Devoluciones: Reingreso al stock o descarte.

Administración y Seguridad

  • Usuarios y Permisos: Sistema de roles granulares.
  • Configuración: Personalización de datos de la empresa (Logo, RIF) para facturas.
  • Backups: Herramienta integrada para copias de seguridad.
  • Logs: Registro de errores mediante NLog.

Tecnologías Utilizadas

C# .NET 8.0 WPF (XAML) SQLite NLog LoremNET
