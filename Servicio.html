Sistema de Gestión de Partituras

Plataforma web para la Estudiantina UNET "Eufrasio Medina"

Laravel 12.x PHP 8.2+ Tailwind CSS 4.x

Este sistema es una aplicación web moderna desarrollada con Laravel diseñada para administrar eficientemente el patrimonio musical de la Universidad Nacional Experimental del Táchira (UNET). Permite la clasificación, préstamo y digitalización del catálogo de partituras de la agrupación fundada en 1976.

Funcionalidades del Sistema

Gestión de Catálogo

  • Partituras: Registro detallado con metadatos de cada obra.
  • Autores e Instrumentos: Clasificación organizada por compositor y tipo de instrumento.
  • Ubicación Física: Gestión de estantes para localizar el material físico.

Sistema & Seguridad

  • Roles: Diferenciación entre Administradores y Usuarios regulares.
  • Préstamos: Control de inventario y seguimiento de material prestado.
  • Códigos QR: Generación automática para acceso rápido a la ficha digital.
  • Interfaz: Diseño responsive con Modo Oscuro/Claro nativo.

Stack Tecnológico

Laravel 12.x PHP 8.2 Tailwind CSS 4.0 Blade Vite SQLite / MySQL Simple QR Code

Instalación y Despliegue

1. Clonar y Configurar

git clone https://github.com/tu-usuario/sistema-partituras-unet.git cd sistema-partituras-unet

2. Dependencias (Backend & Frontend)

composer install npm install

3. Configuración de Entorno

cp .env.example .env php artisan key:generate

4. Base de Datos & Ejecución

# Crear base de datos local touch database/database.sqlite php artisan migrate --seed # Compilar assets y servir npm run dev php artisan serve --port 8050

Contexto Académico

Este proyecto busca preservar y organizar el archivo histórico de la Estudiantina "Eufrasio Medina", facilitando el acceso a músicos y directores a un repertorio de música venezolana andina acumulado durante décadas.
